# CyberSeguridad

# Install Project
1. git clone https://github.com/jordi0907/ModulosCyberseguridad
2. Install nodejs 

# Execute Backend
1. cd backend
2. Execute "npm install" and "npm update". 
4. Execute "npm run dev"

# Execute Frontend

1. cd frontend 
2. Execute "npm install" and "npm update". 
3. Execute "npm run start"
